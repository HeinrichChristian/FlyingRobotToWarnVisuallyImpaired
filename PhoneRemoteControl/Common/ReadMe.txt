Das gemeinsame Verzeichnis enthält Klassen zur Vereinfachung der 
Anwendungsentwicklung. 

Klassen im gemeinsamen Verzeichnis sind Teil Ihres Projekts und können an Ihre 
Bedürfnisse angepasst werden. Gehen Sie beim Ändern bestehender Methoden und 
Eigenschaften vorsichtig vor, da inkompatible Änderungen entsprechende 
Änderungen an Code in einer Vielzahl von Visual Studio-Vorlagen erfordern. 
Beispielsweise werden zusätzliche Ihrem Projekt hinzugefügte Seiten in der 
Annahme geschrieben, dass die ursprünglichen Methoden und Eigenschaften in 
gemeinsamen Klassen immer noch vorhanden sind und dass die Namen der Typen 
nicht geändert wurden.
